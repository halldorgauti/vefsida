Hönnun á vefsíðu

Hér mun ég fara yfir hvernig ég hannaði mína eigin vefsíðu. Við hönnunina á vefsíðunni var forrituð html skrá sem er svo komið á netið með hjálp Git og Github.com. Til einföldunar var fundið fallegt sniðmát á vefsíðunni html5up.net sem kallast Astral. Í möppunni sem fylgir sniðmátinu má finna html skrá sem inniheldur þann kóða sem þarf að endurrita.

Til þess að hanna síðuna að að mínum kröfum var html kóðinn settur upp í forritunarviðmótið "Brackets". Brackets er open-source viðmót sem auðveldar hönnun þegar unnið er að vefsíðum með html forritunartungumálið.

Þar sem að þetta er mín fyrsta vefsíðugerð og notkun html tók það mig nokkurn tíma að ná tökum á tungumálinu. En mér tókst þó að hanna vefsíðuna nokkurn veginn eins og ég hafði hugsað mér. Síðan inniheldur fjóra flipa: Forsíða, ferilskrá, verkefni, hafa samband og facebook.

Til þess að koma vefsíðunni á netið var notast við Git og Github.com. Settur var upp aðgangur að Github og Git forritinu halað niður af netinu. Eftir að hafa stofnað aðgang að Github var búið til nýtt repository á síðunni:

Git er svo notað til að færa öll skjöl frá hörðum disk á tölvu og upp í ský Github.com. Git Bash sett upp í möppu sem inniheldur öll skjöl sem ætlað er upp í skýið.

Framkvæma þarf nokkrar skipanir í Git til þess að koma skjölunum upp í ský á Github. Skipanirnar sem ég framkvæmdi eru í eftirfarandi röð:

1. git init

2. git add .

3. git commit -m "first commit"

4. git branch -M main

5. git remote add origin https://github.com/halldorgauti/vefsida.git

6. git push -u origin main

Eftir að framkvæma þessar skipanir eru öll nauðsynleg gögn komin í skýið og það eina sem eftir er eftir er að koma síðunni í loftið. Það er gert með því að fara í setting á github og velja Branch: main undir Source. Þá hefur síðunni verið komið í loftið á url: https://halldorgauti.github.io/vefsida/
